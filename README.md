# dotfiles
Catppuccin themed NixOS flake for hardcore Rustaceans

![wallpaper](https://github.com/DaraJKong/dotfiles/blob/2219742ee92756f6a0ca7cb1f5d881da06194843/theme/wall.png)

## Features
- 🟢 GNOME + X11 + Nvidia
- 💻 Multiple hosts setup
- 🐱 Catppuccin themed
- 🦀 Everything Rust
